# Подсказки по командной строке

Команда смены директории:
```sh
cd c:\folder_name
```

Команда отображения текущей директории:
```sh
pwd
```
Листинг текущей директории
  
для Windows:
```sh
dir
```
для Linux and Macos:
```sh
ls
```
Удаление файла 

в Windows:
```sh
del <filename>
```
в Linux and Macos:
```sh
rm <filename>
```
